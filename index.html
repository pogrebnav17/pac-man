Instructions

  • To pause/unpause the game press the SPACEBAR
  • Get as many points as you can before the ghosts catch you!
  • To get to the next level you have to get a certain number of points
  • Ghosts speed up with each level!
  • Your Pac-man can run off the board where there is an escape door and return on the other side
  • Cherries are worth 100 points and freeze the ghosts for 3 seconds.
